# Factory Landing — Anajak Website

## Project
Marketing landing page สำหรับโรงงาน Anajak — pillar page SEO, services (DTF/DTG/silkscreen), portfolio, blog

## Business
**Anajak** — เว็บหลักโรงงาน

## Stack
- Framework: Next.js 14 (App Router), React 18, TypeScript
- Styling: Tailwind CSS v3, Framer Motion, Lucide icons
- CMS: Supabase (optional)

## How to Run
```bash
npm install
npm run dev    # localhost:3000
```

## Key Files
- `src/app/page.tsx` — Homepage (pillar page)
- `src/config/site.ts` — Business metadata
- `src/data/services.ts` — Service data
- `src/data/products.ts` — Product data
- `src/data/blog.ts` — Blog content
