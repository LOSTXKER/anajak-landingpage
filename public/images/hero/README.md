# Hero Images

ใส่รูปสำหรับ Hero Section ที่นี่

## รูปที่ต้องการ

| ไฟล์ | ขนาดแนะนำ | คำอธิบาย |
|------|-----------|----------|
| `hero-main.jpg` | 800x1000px (portrait) | รูปหลักสำหรับ Hero - ควรเป็นภาพผลงานเสื้อที่สวยที่สุด |

## หมายเหตุ

- รองรับ `.jpg`, `.jpeg`, `.png`, `.webp`
- ควรเป็นภาพคุณภาพสูง (High Resolution)
- ภาพแนวตั้ง (Portrait) จะดูดีที่สุด
- หากไม่มีรูป จะแสดง placeholder gradient สีฟ้า
