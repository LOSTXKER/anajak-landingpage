# Client Logos

ใส่โลโก้ลูกค้าที่นี่

## รูปที่ต้องการ

| ไฟล์ | ขนาดแนะนำ | คำอธิบาย |
|------|-----------|----------|
| `logo-1.png` | 200x80px | โลโก้ลูกค้า 1 |
| `logo-2.png` | 200x80px | โลโก้ลูกค้า 2 |
| `logo-3.png` | 200x80px | โลโก้ลูกค้า 3 |
| `logo-4.png` | 200x80px | โลโก้ลูกค้า 4 |
| `logo-5.png` | 200x80px | โลโก้ลูกค้า 5 |

| `logo-7.png` | 200x80px | โลโก้ลูกค้า 7 |
| `logo-8.png` | 200x80px | โลโก้ลูกค้า 8 |

## หมายเหตุ

- ใช้ `.png` พร้อมพื้นหลังโปร่งใส (transparent)
- โลโก้ควรเป็นสีดำหรือสีเข้ม (จะแสดง grayscale)
- Hover จะแสดงสีปกติ
- หากไม่มีรูป จะแสดง placeholder สีเทา
